# Hafsa Suleyman Thesis Notebook

## Income Prediction with Fairness Evaluation using Neural Networks

This repository contains the code for my bachelor's thesis for the BSc Cognitive Science and Artificial Intelligence at Tilburg University, analyzing income prediction using neural networks on American Community Survey (ACS) 2024 data, with a focus on racial disparities and fairness evaluation.

## Overview

This project investigates whether machine learning models trained on survey data reproduce or amplify racial wage gaps. It includes:

- Income prediction using a neural network built in PyTorch
- Data preprocessing of ACS 2024 census data
- Analysis of racial disparities across White, Black, and Asian groups
- Adversarial debiasing and probe-based fairness evaluation

## Data

The analysis uses the **American Community Survey (ACS) 2024** dataset publicly available through the Folktables Python package (Ding et al., 2021).. Due to file size, the data is not included in this repository.
